node开发日志功能

Jul 11, 2019

日志需要的文件

不使用第三方日志模块方式

  1. 在utils/log.js文件中定义功能

  2. 在app.js中使用刚才写的日志功能

使用morgan模块方式

  1. 下载morgan模块
    npm i koa-morgan

  2. app.js

    这样就可以了，在每次访问的时候就会在日志中写入一条记录，如果在开发时不需要写入到文件中，可以使用环境变量来区分
